Git常用命令

命令行接口，一切都是在黑窗口中通过命令进行交互

提示：

  • 在命令行中一般只能处理文件或者说是文本文件

  • tab 自动补全，连按两次会将所有匹配内容显示出来

  • 按下上箭头可以调出历史记录
  • 操作系统有用户的功能，可以创建多个不同的用户
  • 不同的用户具有自己的用户目录，一般存在于 c:/Users/用户名
  • 在终端中，目录中的 ~ 一般就标识用户目录
  • 当前工作目录
